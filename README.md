# 🎯 Number Guessing Game

A simple and interactive Number Guessing Game built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic web development concepts and is suitable for hosting on **GitHub Pages or AWS (S3 / EC2)**.

## 📌 Features
- Guess a number between **1 and 100**
- Limited number of attempts
- Timer-based gameplay
- Reset game functionality
- Clean and responsive UI
- Beginner-friendly JavaScript logic

---

## 🛠 Technologies Used
- **HTML5** – Structure
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Game logic
- **Git & GitHub** – Version control
- **Terminal (macOS)** – File handling & setup
